REST Countries Explorer

A solution to the Frontend Mentor REST Countries API challenge built with React.

Overview

This project allows users to browse countries around the world, search by name, filter by region, and view detailed information about each country.

Users can:

- Search countries by name
- Filter countries by region
- View detailed country information
- Navigate through bordering countries
- Switch between light and dark mode
- Open a country location in Google Maps
- Use the application on desktop and mobile devices

Features

Search and Filter

Users can search countries by name and filter the results by region.

Country Details

Each country includes:

- Native name
- Population
- Region
- Sub region
- Capital
- Top level domain
- Currency
- Languages

Border Countries

Border countries are displayed as clickable buttons, allowing users to navigate between neighboring countries.

Google Maps Integration

Each country includes a link that opens its location in Google Maps.

Dark Mode

Users can switch between light and dark themes.

Responsive Design

The layout adapts to different screen sizes, providing a mobile-friendly experience.
